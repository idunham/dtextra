Meditres

Meditres is a Motif port of Editres2. Editres2 has support for reading as well as writing resource values, meditres inherits that.

The user interface of meditres is based on Motif (LessTif), currently the widget tree is visualised using XmContainer.

As editres carries the MIT license, so does meditres.
