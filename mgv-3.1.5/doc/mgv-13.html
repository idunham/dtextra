13. The Tool Bar

The mgv tool bar provides convenient access to various functions which are available via the menus. The functions available from the tool bar are currently hard wired, there is no way add or remove buttons without chaning the mgv source code. However, various visual aspects of the tool bar can be configured through the resource database.

Users can change the icons used in the tool bar by setting the toolBarIcon resource on the tool bar buttons. If a tool bar icon cannot be found, the built-in default images (Dobbs heads) will be used and a warning message will be printed to the standard error stream.
