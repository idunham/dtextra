12. The Popup Menu

A popup menu is available anywhere in the main window by pressing mouse button three; this menu contains the entries in the View menu with the following additions:

Back (Alt+Left)

Go to the previous page in the page history list; the previous page is the page you were at before the current page.

Forward (Alt+Right)

Go forward in the page history list.

The Forward and Back commands behave like Forward and Back in most web browsers; the only difference is that the page history list is initialized to a list of all the pages in the current document. There is currently no way to view the page history list, if you really want such a feature send a request to the mgv author.
